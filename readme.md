## Usage

1. Datenbank Starten  <br>
A) Falls Path Variablen Vergeben,  Starte ```db.cmd```

2. Server Dependencies Installieren <br>
A) Befehl: ```npm install```
                                                         
3. Server Starten (Befehl:  node index.js) <br>
A) Befehl: ```node index.js```
